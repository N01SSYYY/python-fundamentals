What is a dictionary?
A dictionary is a mutable collection of key–value pairs, where each key maps to a value.

Difference between list and dictionary
A list stores values in an ordered sequence and is accessed by index.
A dictionary stores values mapped to unique keys and is accessed by key.

Why keys must be unique
Keys must be unique because each key maps to exactly one value.
If keys were duplicated, Python would not know which value to retrieve.

One real-world use-case
Dictionaries are used to store structured data, such as mapping student names to their marks.
